# My Resume Django Project

This is a Django project created to showcase my resume in a professional and interactive format. It leverages Python for backend logic and Django for the web framework.

**Features**

* **Professional layout:** A visually appealing and well-structured design.
* **Dynamic content:** Use of Django templates and Python to easily update and customize your resume.
* **Responsive design:** Looks great on various devices (desktop, tablet, mobile).
* **SEO optimization:** Basic SEO elements to improve search engine visibility.

**Technologies used**

* Django
* Python
* HTML
* CSS

**How to run the project**

1. Clone the repository from GitHub.
2. Install the requirements (including Python and Django).
3. Create a database.
4. Migrate the database.
5. Run the server.

**Dependencies**

* Django
* Python
